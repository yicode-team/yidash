# static tree_Array2Tree(arrs, id, pid, children) → {Array}

应用场景：用于生成无限级菜单结构

一维数组生成无限级树结构
Parameters:
Name Type Description
arrs Array 传入的一维数组
id String 唯一标识字段
pid String 父级标识字段
children String 子级标识字段

View Source tree.js, line 52

返回一个无限级数组结构
Array
Example
import { tree_Array2Tree } from 'yidash';
tree_Array2Tree([
     { id:1, pid:0, name:'a' ,age:16 },
     { id:2, pid:1, name:'b' ,age:18 },
     { id:3, pid:2, name:'c' ,age:22 },
     { id:4, pid:2, name:'d' ,age:25 },
])
// 结果如下：
[{
     id:1,
     pid:0,
     name:'a',
     age:16,
     children:[{
         id:2,
         pid:1,
         name:'b',
         age:18,
         children:[
             {
                 id:3,
                 pid:2,
                 name:'c',
                 age:22,
                 children:[]
             },
             {
                 id:4,
                 pid:2,
                 name:'d',
                 age:25,
                 children:[]
             }
         ]
     }]
}]
